# DAMRU 2000 MODEL VAULT

## Inventory
- 100 real/current/recent/historical/reference aircraft families × 10 parametric configurations = 1000 aircraft presets.
- 50 commonly used 3D archetypes × 20 configurations = 1000 general presets.
- Total: 2000 unique executable preset IDs.

## Important honesty note
These are executable parametric presets that generate editable scene graphs. They are not downloaded OEM meshes, exact classified geometry, airworthy CAD, or manufacturer-certified dimensions. Concept aircraft are explicitly named `concept`.

## Identity routing
`F-35`, `B-21`, `A321XLR`, `C919`, `YFQ-42A`, etc. are matched before the generic aircraft branch. A Vault baseline appears first; Damru Deep Build may refine it only when the quality gate accepts the result.

## UI
Open Visualise and tap `🗃 2000 Vault`. Search all entries, filter Aircraft/General, or use Surprise. Only the first 120 matching cards render for mobile performance; search covers all 2000.

## Included preserved systems
Universal Forge, 404-part DNA, molecule/protein compilers, Starship compiler, colour/material editor, Truth Forge, Universe Sim, SimLab and Print Forge.

## Upload
Extract the ZIP and replace/add all files together in the GitHub repository root. `index.html` already loads `damru_model_vault.js` before `damru_visualise.js`.
