# DAMRU UNIVERSAL FORGE + UNIVERSE SIM

## New build
- Deterministic 28-base-pair B-DNA double helix with two sugar-phosphate backbones, editable A–T/G–C colours, covalent/hydrogen bonds and 10.5 bp/turn.
- Molecular ball-and-stick compiler for water, generic methane-like tetrahedral carbon, CO2 and benzene.
- Coarse alpha-helix protein compiler with residues, peptide links and side chains.
- Offline compilers for robots, cars, aircraft, trees, solar systems, architecture, gears, furniture and spacecraft.
- Unknown prompts receive a coloured multi-part semantic assembly—never the old `base + concept-core` cylinder.
- Every non-empty prompt uses Damru Deep Build: web research, multidisciplinary brief, synthesis, repair and no-downgrade gate. Worst-case timeout envelope is about 2–3 minutes; actual time depends on backend/network.
- Arbitrary endpoint `bond` geometry for scientific structures.
- Selected-part colour and roughness are live-editable; GLB preserves materials.
- Universe Sim transfer: Earth, Moon, Mars, orbital vacuum, underwater, molecular and biological environments.
- Coarse rigid dynamics: gravity, drag, wind/fluid force, buoyancy, ground collision, bounce and semi-implicit Euler integration.
- Chemistry/biology modes: explicit coarse harmonic-bond and thermal-motion visualization.
- Damru AI preflight reasons about assumptions, equations, applicable domains, failure modes and required tests.

## Honest scope
No text-only system can guarantee perfect CAD for literally every object without references, dimensions and constraints. Universe Sim is an interactive coarse digital twin—not certified FEM/CFD, quantum chemistry, protein-fold prediction, clinical evidence, or a substitute for physical testing.
